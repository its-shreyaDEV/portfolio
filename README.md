🏮 Neon Dhaba Portfolio
Welcome to my digital Indian Dhaba—where high-speed code meets vibrant neon aesthetics. This is a creative portfolio website built to showcase my projects with a "Desi-Modern" twist, blending traditional dhaba vibes with futuristic neon elements.

✨ Features
Neon Desi UI: A dark-mode interface with glowing neon accents (pink, saffron, and electric teal).

Menu-Style Navigation: Browse projects like a dhaba menu card—categorized and "tasty."

Responsive Layout: Hearty and flavorful viewing on mobile, tablet, and desktop.

Interactive Truck Art: Subtle animations inspired by vibrant Indian truck art patterns.

🛠️ Tech Stack
Frontend: React / Next.js (or your chosen framework)

Styling: Tailwind CSS / CSS Modules (with custom Glow effects)

Animations: Framer Motion / GSAP

Icons: Lucide React / Custom SVG Truck Art

🚀 Quick Start
To get this "dhaba" running locally, follow these steps:

Clone the repo:

Bash
git clone https://github.com/its-shreyaDEV/portfolio.git
Install dependencies:

Bash
npm install
Start the kitchen:

Bash
npm run dev
🎨 Design Philosophy
The design is inspired by the roadside dhabas of India—rustic, welcoming, and full of life—reimagined for the 2026 digital age. It uses glassmorphism to mimic roadside signage and vibrant gradients to reflect the rich colors of Indian spices and neon lights.
